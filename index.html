Rencontre #3: Utilisation de bases de données avec SQL et interface avec R

SQL est un langage particulier de programmation utilisé avec les bases de données. Nous verrons comment lire, trier et filtrer des données, calculer de nouvelles valeurs à la volée, gérer les données manquantes, agréger des données, combiner des informations provenant de plusieurs tables, etc grâce au langage SQL. Nous verrons aussi comment accéder à une base de données à partir de R. Pour ce faire nous utiliserons un gestionnaire de bases de données simple appelé SQLite, et ce à partir d'un plugin du navigateur internet. Vous aurez besoin d'installer le(s) programme(s) ci-dessous. Si vous rencontrez des difficultés, prenez contact avec l'organisateur ou venez plus tôt à l'atelier.

Choisissez la date de la prochaine rencontre en votant ici.

Windows

Télécharger l'installateur de Software Carpentry. Si le fichier s'ouvre directement dans votre navigateur, choisir Fichier→Sauver la page comme pour le télécharger sur votre ordinateur. Double-clicker sur le fichier pour l'éxécuter. Nous utiliserons le navigateur internet Firefox, auquel vous ajouterez l'extension SQLite décrite ci-dessous. Si vous avez participé à l'atelier Software Carpentry, vous n'avez pas besoin d'installer l'installateur.

Mac OS X

sqlite3 vient pré-installer sur Mac OS X. Vous utilisez probablement déjà Firefox, auquel vous ajouterez l'extension SQLite décrite ci-dessous.

Linux

sqlite3 vient pré-installer sur Linux. Vous utilisez déjà Firefox, auquel vous ajouterez l'extension SQLite décrite ci-dessous.

Firefox SQLite Plugin

Au lieu d'utiliser sqlite3 en ligne de commande, nous utiliserons cette extension pour Firefox. Pour l'installer:

  • Démarrer Firefox.
  • Aller à la page du plugin.
  • Clicker sur le bouton "Add to Firefox".
  • Clicker "Installer maintenant" dans la boîte de dialogue qui apparaît after que le téléchargement soit complété.
  • Redémarrez Firefox quand c'est demandé.
  • Pour vérifier que le plugin est bien installé, sélectionnez "SQLite Manager" à partir du menu "Outils".

Dans les nouvelles versions de Firefox, la barre de menu n'est pas toujours visible. Pour la faire apparaître, utiliser la touche Alt à côté de la barre d'espacement sur votre clavier, ou consulter la page de support de Firefox pour de l'aide supplémentaire.
