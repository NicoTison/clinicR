Rencontre #2: Markdown et R Markdown pour des documents "dynamiques"

R Markdown est un format qui facilite la création de rapports reproductibles à partir de R. Il combine la syntaxe Markdown (un format texte brut facile à écrire) à des pièces de code R imbriquées et exécutées de sorte que leurs résultats peuvent être inclus dans le document final.
